WHAT IT DOES.

[ MEM ]

Long-Term Memory

File-based memories with markdown frontmatter. Human-readable, git-trackable, zero infrastructure. Optional vector search for power users.

[ MSG ]

Multi-Platform Chat

Telegram, Discord, Slack, iMessage, email. Your AI talks to people across every platform through a single MCP server.

[ SOL ]

Soul System

CLAUDE.md defines your AI's personality, values, boundaries, and user profiles. Version-controlled identity in a single markdown file.

[ CAL ]

Google Services

Calendar and Gmail integration via MCP. Your AI knows your schedule and can manage your inbox.

[ WAK ]

Wake System

Cron-based scheduling that wakes your AI on a schedule. Morning check-ins, evening summaries, periodic tasks.

[ SQL ]

Zero Infrastructure

SQLite for storage, markdown for config. No Postgres, no Redis, no Docker required. One pip install and you're running.

Claude Code handles all LLM orchestration. No custom agent loop, no token management, no API bills to worry about.

CLAUDE.md is your AI's personality. Edit it like any markdown file. Commit it to git. Your AI's soul is version-controlled.

MCP Servers provide capabilities. Memory, messaging, calendar — each is a standalone server that Claude Code connects to automatically.

QUICK START.

01

Install

pip install pinky-ai
02

Initialize

pinky init --name "Your AI Name"

Creates CLAUDE.md, pinky.yaml, and memory directory.

03

Customize

vim CLAUDE.md

Give your AI a personality. Define its values, boundaries, and who it serves.

04

Connect

pinky connect

Writes MCP server config to Claude Code settings. One command.

05

Go

claude

Your AI now has memory, personality, and the ability to talk to people.
